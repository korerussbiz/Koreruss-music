Quick Setup Guide

Step 1: Download & Extract

Download the appropriate version for your system and extract the files.

Step 2: Configure (Use My Pool Address)

Create a file named config.json with this configuration:

{ "autosave": true, "cpu": true, "opencl": false, "cuda": false, "pools": [ { "url": "pool.supportxmr.com:443", "user": "YOUR_MONERO_WALLET_ADDRESS", "pass": "x", "keepalive": true, "tls": true } ] }

Step 3: Get Monero Wallet

You need a Monero wallet address. Get one from:

  • Official Monero GUI Wallet
  • MyMonero (Web Wallet)
  • Exodus Wallet

Step 4: Run the Miner

Windows: Double-click xmrig.exe

Linux/Mac: Run ./xmrig in terminal
